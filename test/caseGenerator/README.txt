* Introduction:
  =============
  This module is used to genarate testcase form tables in word document.

* Dependency:
  =============
  You should install python-lxml first. The command is "sudo apt-get install python-lxml"

* Shell command:
  ============= 
  python genCase.py "${MOS_HOME}/testnew/caseGenerator/NewTestCase.docx" "${MOS_HOME}/testnew/testcases/casemodel/testcase.yml"
